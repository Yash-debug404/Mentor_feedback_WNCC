# Mentor_feedback_WNCC

## HOW TO RUN:

1. First clone the repository:
   
   git clone git@github.com:Yash-debug404/Mentor_feedback_WNCC.git
   cd Mentor_feedback_WNCC

2. Run main.py script:
   
   python3 main.py
